Daily Briefing App
A Python application that provides a personalized daily briefing by combining the latest news and current weather information.
Features
* Fetches top 5 news headlines using NewsAPI
* Displays news sources
* Fetches current weather using OpenWeatherMap API
* Shows temperature and humidity
* Secure API key storage using `.env`
Installation
pip install -r requirements.txt
Run
python app.py
Technologies Used
* Python
* Requests
* Python Dotenv
* NewsAPI
* OpenWeatherMap API
